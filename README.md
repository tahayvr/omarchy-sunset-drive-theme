# Sunset Drive

Theme for [Omarchy Linux](https://omarchy.org) based on a synthwave aesthetic.

Made with [Omarchist](https://github.com/tahayvr/omarchist).

## Installation

```bash
omarchy-theme-install https://github.com/tahayvr/omarchy-sunset-drive-theme
```
